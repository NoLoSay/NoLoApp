## Install

```sh
yarn install
```

## Usage

First launch your metro :

```sh
yarn start
```

Then launch the app on your device, depending on the wanted OS :

```sh
yarn ios
yarn android
```

## Tests

Tests are done with [Jest](https://jestjs.io/)

To launch them :

```sh
yarn test
```
